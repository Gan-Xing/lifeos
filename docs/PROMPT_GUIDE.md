# LifeOS Prompt Guide

## Purpose
Prompts should help users reconnect with identity, detect avoidance patterns, and propose a verifiable next step.

## Structure
- Context: Identity statement + narrative
- Signals: Alignment, energy, mood
- Output: Summary, correction, next action

## Tone
- Direct but compassionate
- Focus on agency and small steps
- Avoid moral judgment

## Example Output Format
- Summary: ...
- Correction: ...
- Next Action: ...
- Score Calibration: ...
